# Universal Video Speed Controller

**Universal Video Speed Controller** is a browser extension that allows you to control the playback speed of any HTML5 video on any website. Watch videos at your own pace—slow down, speed up, or reset with ease!

## Features

- Increase or decrease video playback speed in real-time.
- Reset speed to normal with a single click or keyboard shortcut.
- Set a fixed speed for consistent viewing.
- Keyboard shortcuts for quick speed control.
- Works on any website with HTML5 videos.
- Lightweight and easy to use.

## Installation

1. **Download the extension** folder from this repository.
2. Open your browser and go to the **Extensions** page:
   - Chrome: `chrome://extensions/`
   - Edge: `edge://extensions/`
   - Firefox: `about:addons`
3. Enable **Developer Mode** (Chrome/Edge) or **Debug Mode** (Firefox).
4. Click **Load unpacked** and select the extension folder.
5. The extension icon will appear in your browser toolbar.

## Usage

1. Click the extension icon in the toolbar to open the popup.
2. Configure your **Speed Step**, **Reset Speed**, and **Fixed Speed** values.
3. Set your preferred **keyboard shortcuts** for increasing or decreasing speed.
4. Toggle the extension **on or off** as needed.
5. Play any HTML5 video—use the popup or shortcuts to control playback speed in real-time.

## Keyboard Shortcuts

- **Increase Speed**: configurable in the popup (default key `]`)
- **Decrease Speed**: configurable in the popup (default key `[`)
- **Fixed Speed**: configurable in the popup (default key `G`, not configurable)
- **Reset Speed**: fixed value (`1`) (default key `R`, not configurable)

## Permissions

- `storage` – to save your speed settings and keyboard shortcuts.
